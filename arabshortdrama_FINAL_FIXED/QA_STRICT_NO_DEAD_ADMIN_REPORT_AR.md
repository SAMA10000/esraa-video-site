# تقرير QA صارم — إزالة التبويبات الميتة وتحويلها لوظائف حقيقية

## الحكم قبل الإصلاح
الصور أظهرت أقسامًا داخل لوحة الأدمن تحتوي على نص عام وزر تحديث فقط. هذا كان FAIL لأنه يخالف قاعدة: كل تبويب شغال بباك إند حقيقي أو يتم إزالته.

## ما تم تنفيذه كـ Minimal Fix

### 1) تبويبات الأدمن
- تم إلغاء شريط التبويبات الأفقي المكسور واستبداله بـ flex-wrap حتى لا يظهر scrollbar مزعج.
- تم حذف أي نصوص من نوع: سيتم ربط API لاحقًا / واجهة مستقرة / استخدمي زر التحديث.
- لا توجد أي أقسام روابط تفعيل أو شركاء أو allowlist.

### 2) إدارة المسلسلات
- إضافة مسلسل من MySQL API.
- تعديل مسلسل من MySQL API.
- حذف مسلسل من MySQL API.
- التحقق من Dailymotion ID فقط.

### 3) إدارة الفيديوهات
- عرض Dailymotion ID لكل مسلسل.
- تعديل Dailymotion ID وحفظه من MySQL API.
- منع iframe خام؛ الباك إند يقبل ID فقط.

### 4) التصنيفات
- إضافة تصنيف.
- تعديل تصنيف.
- تعطيل/تفعيل تصنيف.
- حذف تصنيف فقط لو لا يحتوي على مسلسلات.

### 5) البوسترات والواجهة
- عرض بوسترات 9:16.
- تعديل رابط البوستر وحفظه من MySQL API.

### 6) التحليلات
- تم إنشاء endpoint جديد: api/public/analytics.php.
- تم ربط تتبع الصفحة، فتح المسلسل، ظهور المودال، CTA، الصوت، والشات بـ analytics_events.
- لوحة التحليلات تقرأ من MySQL وليس أرقامًا وهمية.

### 7) الدردشة والمجتمع
- عرض أسماء محجوزة من MySQL.
- عرض آخر رسائل الشات من MySQL.
- تحرير اسم محجوز من الباك إند.
- حفظ إعدادات الدردشة داخل admin_settings.

### 8) إعدادات البوتات
- حفظ knowledgeBase و targetGame و steeringWeight و feedEnabled و pollingInterval في MySQL.
- لا يعتمد على localStorage للحفظ النهائي.

### 9) إدارة المسؤولين
- إنشاء مدير من الباك إند.
- تغيير صلاحية مدير.
- تفعيل/تعطيل مدير.
- حماية آخر super_admin من التعطيل أو إنزال الصلاحية.

### 10) الأمان والجلسات
- تسجيل الدخول يستخدم PHP session cookie.
- تم تفعيل جدول admin_sessions عند login/logout/session/password-change.
- تغيير كلمة المرور يحذف جلسات الأدمن.
- Storage Security Audit موجود داخل اللوحة.

### 11) صحة النظام
- فحص اتصال قاعدة البيانات.
- فحص الجداول المطلوبة.
- فحص مسارات auth/chat/analytics/dailymotion كحالة تشغيل.

### 12) Installer
- تم تعديل install.php ليشغل seed_mysql.sql بالكامل، وليس التصنيفات فقط.
- seed_mysql.sql يحتوي على 9 categories و 22 dramas وإعدادات admin_settings.
